What is the web?

The Web is the common name for the World Wide Web, a subset of the Internet consisting of the pages that can be accessed by a Web browser.

What is HTML?

HTML stands for => HyperText Transform language. It is the fundamental building blocks of the web page

What is CSS?

CSS stands for => Cascading Style Sheets. It is the the way of styling the web page (HTML elements)

What is JS?

JS stands for => JavaScript. It is a programming language that make the web page more interacive
